# 知识库问答系统用户手册

## 1. 系统简介

知识库问答系统是一个基于大语言模型（LLM）的智能问答服务，支持用户上传文档构建知识库，并通过自然语言提问获取相关知识库中的答案。系统提供简单易用的界面，适用于个人和企业用户。

## 2. 快速开始

### 2.1 登录系统

1. 打开浏览器，访问系统入口地址（如 `https://127.0.0.1:19002`）。
2. 输入用户名和密码，点击“登录”按钮。

### 2.2 上传文档

1. 登录后，点击“上传文档”按钮。
2. 选择需要上传的文档（支持 `.txt`、`.docx` 等格式）。
3. 点击“确认上传”按钮，系统将自动处理文档并构建知识库。

### 2.3 提问与获取答案

1. 在输入框中输入您的问题（如“什么是知识库？”）。
2. 点击“发送”按钮，系统将返回相关知识库中的答案。

## 3. 功能详解

### 3.1 知识库管理

- **上传文档**：支持批量上传文档，系统自动解析并存储。
- **删除文档**：可删除不需要的文档或知识库。
- **查看文档列表**：显示当前用户上传的所有文档。

### 3.2 问答功能

- **自然语言提问**：支持用自然语言提问，系统返回相关答案。
- **多轮对话**：支持上下文关联的多轮问答。
- **答案来源**：显示答案对应的文档片段，方便用户验证。

### 3.3 用户设置

- **创建**：支持用户创建账户。
- **会话管理**：显示当前登录会话信息，支持手动退出。

## 4. 常见问题

### 4.1 上传文档失败

- **原因**：文档格式不支持或文件过大。
- **解决方案**：检查文档格式是否符合要求（如 `.txt`、`.docx`），或联系管理员。

### 4.2 问答结果不准确

- **原因**：知识库中缺乏相关内容或问题表述不清晰。
- **解决方案**：尝试重新表述问题，或上传更多相关文档。

### 4.3 系统响应慢

- **原因**：网络延迟或服务器负载过高。
- **解决方案**：检查网络连接，或稍后再试。

## 6. 附录

### 6.1 支持的文档格式

- 文本文件（`.txt`）
- Word 文档（`.docx`）

### 6.2 系统限制
系统未设定明确的文档个数及大小限制，为了能够有较好的体验，建议
- 单个文档大小不超过 10MB。
- 每个用户最多可上传 100 个文档。
