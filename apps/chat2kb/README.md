# 知识库问答

通过构建自定义知识库，实现基于知识库的 AI 问答服务。

## 支持的文件格式

| 格式 | 说明 |
|------|------|
| `.txt` | 纯文本文件 |
| `.pdf` | PDF 文档 |
| `.docx` | Word 文档 |
| `.xmind` | XMind 思维导图（支持业务流程、层级结构等场景的问答） |

## XMind 文件处理

上传 `.xmind` 文件后，系统会自动解析思维导图的层级结构，将每条从根节点到叶子节点的路径转换为独立的文本片段。支持提取：

- 主题标题及层级关系
- 备注/说明（notes）
- 标签（labels）
- 标记（markers，如优先级、完成状态）

这使得对业务流程、审批流、组织架构等以思维导图形式组织的内容进行自然语言问答成为可能。
